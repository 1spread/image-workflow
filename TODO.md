# TODO / Future Work

## 画像編集機能（未着手）

拡大オーバーレイに編集モードを追加する。元画像は触らず、編集結果はクリップボード/ダウンロード経由で出力する。

### 機能
- モザイク（ドラッグ矩形範囲をピクセル化）
- 赤枠（ドラッグ矩形に枠線）
- 文字入れ（クリック→input→配置）
- 切り取り（ドラッグ矩形でクロップ）
- 矢印（ドラッグで線+三角ヘッド）
- Undo / Done / Cancel

### UI
- Edit ボタンをツールバーに追加
- 編集モード中は Canvas に画像描画 + ツールバー切替
- Done → 編集結果が拡大ビューに反映、Copy / Download / Cmd+C で出力可能

### 実装方針
- 自前 Canvas 実装（ライブラリ依存なし、〜500 行想定）
- Fabric.js 等は bundle が重くなるため避ける
